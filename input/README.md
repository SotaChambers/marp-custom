# inputディレクトリの使い方

このディレクトリには `/convert-to-marp` で変換するテキスト素材を置きます。

## 使い方

1. `template.md` をコピーして内容を書く
2. `/convert-to-marp` で Marpスライドに変換
3. 生成されたスライドを編集・調整

## 書き方のルール

| 書き方 | 変換先 |
|--------|--------|
| `# 見出し` | スライド区切り + タイトル |
| `## 見出し` | セクション区切り（top） |
| `### 見出し` | サブ見出し |
| `ポイント: テキスト` | kg-callout |
| `注意: テキスト` | kg-callout warning |
| `項目名: 値` | kg-spec |
| 左右の対比リスト | kg-two-col |
| `#### タイトル` + リスト | kg-card |

## 画像について

画像は `imgs/` ディレクトリに直接配置してください。
