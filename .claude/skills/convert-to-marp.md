# Marp形式への変換

テキスト資料、会議メモ、箇条書きなどをMarpスライドに変換します。

## 変換ルール

### スライド構成
1. 最初のスライドは title クラス（カバー）
2. 大きなトピック変更時は top クラス（セクション区切り）
3. それ以外は main（デフォルト）

### コンポーネント自動選択

| 入力パターン | 適用コンポーネント |
|-------------|-------------------|
| 「ポイント：」「注意：」で始まる文 | kg-callout |
| 警告・注意事項 | kg-callout warning |
| 2つの対比・比較 | kg-two-col |
| 表形式データ | kg-table |
| ラベル: 値 の形式 | kg-spec |
| 番号付きセクション（1. 〇〇 2. △△） | kg-info-panel |
| カテゴリ+詳細リスト | kg-card |
| タグ・ステータス表示 | kg-badge |

### 情報量の目安
- 1スライドに箇条書きは5項目以内
- 多い場合は複数スライドに分割
- または kg-two-col で2カラム化

## フロントマター

```yaml
---
marp: true
theme: theme
class: title  # 最初のスライドのみ
size: 16:9
footer: "(C) KG motors INC. All Rights Reserved."
---
```

## 指示

1. ユーザーから変換元のテキストを受け取る
2. 内容を分析してスライド構成を決定
3. 適切なコンポーネントを選択
4. Marp形式のマークダウンを生成
5. 必要に応じて画像の配置場所を提案（プレースホルダーとして）

## 出力形式

変換後のマークダウンを完全な形で出力する。
コメントで各スライドの意図を説明する。

```markdown
---
marp: true
theme: theme
class: title
size: 16:9
footer: "(C) KG motors INC. All Rights Reserved."
---

<!-- SLIDE 1: カバー -->
# タイトル
...
```
