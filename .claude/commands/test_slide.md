# スライドの品質チェック

マークダウンスライドの品質をチェックします。

## チェック項目

### 1. フロントマター
- `marp: true` が設定されているか
- `theme` が正しく指定されているか
- `size: 16:9` が設定されているか

### 2. テーマ・画像ファイル
- 指定されたテーマファイル（theme/theme.css）が存在するか
- 参照している画像ファイルが存在するか

### 3. クラス指定
- `<!-- class: xxx -->` の指定が有効な値（title, top, または空）か

### 4. コンポーネント構造
- `kg-two-col` 内に `.col` が含まれているか
- `kg-spec` 内に `kg-spec-label` と `kg-spec-value` があるか
- `kg-info-panel` 内に `kg-info-panel-label` と `kg-info-panel-content` があるか
- `kg-card` 内に `kg-card-header` と `kg-card-body` があるか

### 5. スライド区切り
- `---` でスライドが正しく区切られているか

## 指示

1. 対象のマークダウンファイルを読み込む
2. 上記のチェック項目を順に確認
3. 問題があれば具体的な箇所と修正案を提示
4. すべてOKなら「品質チェック完了」と表示
